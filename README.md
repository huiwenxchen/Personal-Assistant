# Personal-Assistant
React app to help users make a todo list and check today's weather

Install Node.js before running the app. <br/>
View the documentation on https://nodejs.org/en/download/ <br/>

Run these commands in your terminal:<br/>


    $ npm i
    $ npm run dev


If you are on a local host, the react app will open on the 3000 port. The images below show a demo of react todo app:
